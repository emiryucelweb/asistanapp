# AsistanApp Demo Assets

This folder contains static assets for the demo site:

- Images
- Icons
- Fonts
- Other static files

## Usage

Reference files from this folder in your components:

```tsx
<img src="/assets/logo.png" alt="Logo" />
```

## Organization

Suggested structure:
```
assets/
├── images/
│   ├── screenshots/
│   ├── icons/
│   └── logo/
├── fonts/
└── videos/
```
